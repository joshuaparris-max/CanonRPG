# Adding Official Content

This repository is designed for private local preparation using sourcebooks you own.

## Official PDF workflow

- Import PDFs locally on the Sourcebook screen.
- The app indexes PDF text and builds local page suggestions.
- Do not commit PDFs, extracted text, or copyrighted content.

## Safe content

- Use `Export Sourcebook References` for safe export of source references only.
- Keep private DM prep exports separate from shared exports.
